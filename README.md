# Konfhub_BE
BackEnd for Konfhub
## Tasks completed: 
* Printed the contents in a human readable format, e.g.: “San Antonio CyberSecurity Conference”,  November 6th, 2019, San Antonio, TX, US, Paid. https://futureconevents.com/events/san-antonio/”   
* Identified exact duplicates (if any)
* Identified semantic duplicates (i.e., the conferences are same but the details provided are slightly different, e.g., “React Conference 2019” in one entry and “ReactConf ‘19” in another entry but the other fields are same or similar).
